# A股量化交易系统 - 数据获取层规划

## 目标
实现从本地通达信数据源读取股票和指数数据，支持日线和分钟级数据（1分钟、5分钟）。

## 数据源分析

### 通达信本地数据目录结构
```
{TDX_DATA_ROOT}/vipdoc/
├── sh/lday/      # 上海市场日线数据 (.day)
├── sh/minline/   # 上海市场1分钟数据 (.lc1)
├── sh/fzline/    # 上海市场5分钟数据 (.lc5)
├── sz/lday/      # 深圳市场日线数据 (.day)
├── sz/minline/   # 深圳市场1分钟数据 (.lc1)
├── sz/fzline/    # 深圳市场5分钟数据 (.lc5)
└── bj/lday/      # 北京交易所日线数据 (.day)
```

### 通达信日线数据 (.day) 格式 - ✅ 已验证
**每记录32字节，小端序(Little Endian)**

| 字段       | 偏移 | 长度 | 类型  | 说明                    |
|-----------|------|------|-------|------------------------|
| 日期       | 0    | 4    | uint32 | YYYYMMDD 格式           |
| 开盘价     | 4    | 4    | uint32 | 精确到分，乘以100       |
| 最高价     | 8    | 4    | uint32 | 精确到分，乘以100       |
| 最低价     | 12   | 4    | uint32 | 精确到分，乘以100       |
| 收盘价     | 16   | 4    | uint32 | 精确到分，乘以100       |
| 成交量     | 20   | 4    | uint32 | 手数                    |
| 成交额     | 24   | 4    | uint32 | 元                      |
| 保留字段   | 28   | 4    | uint32 | 备用                    |

### 通达信分钟数据 (.lc1/.lc5) 格式 - ⚠️ 需要进一步分析
**每记录32字节**

| 字段       | 偏移 | 长度 | 类型    | 说明                    |
|-----------|------|------|---------|------------------------|
| 字段0     | 0    | 4    | uint32  | 未知（可能包含时间信息但编码特殊） |
| 开盘价     | 4    | 4    | float   | 浮点数格式，直接使用     |
| 最高价     | 8    | 4    | float   | 浮点数格式，直接使用     |
| 最低价     | 12   | 4    | float   | 浮点数格式，直接使用     |
| 收盘价     | 16   | 4    | float   | 浮点数格式，直接使用     |
| 成交额     | 20   | 4    | uint32  | 元                      |
| 成交量     | 24   | 4    | uint32  | 手数                    |
| 保留字段   | 28   | 4    | uint32  | 备用                    |

**注意**：分钟数据的浮点数格式与小端序int解析兼容，例如int值1152475955解析为float 1419.10元。

### 股票代码规则
- **上海：** `sh` + 6位代码（如 `sh600000`）
- **深圳：** `sz` + 6位代码（如 `sz000001`）
- **北京：** `bj` + 6位代码（如 `bj830933`）

## 数据模型

### StockData 数据类
```python
@dataclass
class StockData:
    code: str           # 股票代码，如 "000001"
    name: str           # 股票名称（可选）
    market: str         # 市场，如 "sz", "sh", "bj"
    freq: str           # 频率，如 "daily", "1min", "5min"
    data: pd.DataFrame # OHLCV 数据

DataFrame 包含列：
- date: datetime64
- open: float
- high: float
- low: float
- close: float
- volume: int
- amount: float (仅日线数据)
```

## API 设计

### 核心接口

```python
class TdxDataReader:
    def __init__(self, data_root: Optional[str] = None):
        """初始化数据读取器

        Args:
            data_root: 通达信数据根目录，默认从环境变量 TDX_DATA_ROOT 读取
        """

    def get_stock_data(
        self,
        code: str,
        start_date: Optional[str] = None,
        end_date: Optional[str] = None,
        freq: str = "daily"
    ) -> StockData:
        """获取单只股票数据

        Args:
            code: 股票代码，如 "000001" 或 "600000"
            start_date: 开始日期，格式 "YYYY-MM-DD"
            end_date: 结束日期，格式 "YYYY-MM-DD"
            freq: 频率，"daily", "1min", "5min"

        Returns:
            StockData 对象
        """

    def get_index_data(
        self,
        code: str,
        start_date: Optional[str] = None,
        end_date: Optional[str] = None
    ) -> StockData:
        """获取指数数据，如 "000001"（上证指数）"""

    def list_available_stocks(
        self,
        market: str = "all",
        freq: str = "daily"
    ) -> List[str]:
        """列出可用的股票代码

        Args:
            market: 市场筛选，"all", "sh", "sz", "bj"
            freq: 频率，"daily", "1min", "5min"
        Returns:
            股票代码列表
        """
```
